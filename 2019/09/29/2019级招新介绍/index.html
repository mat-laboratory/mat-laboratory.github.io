关于实验室

我们实验室分为前端、后台、安卓、硬件（物联网）、大数据五个方向，接下来我们将会介绍每个方向的具体内容和一些特别的要求。（以下内容由各部门负责人撰写）

前端

前端是整个应用的颜面，他负责了整个应用颜值，尽最大努力发挥自己的想像，你可以创造一个独特的应用，在这里你可以看到美丽的学姐，帅气的学长，可以先其它部门一步感受到做出软件的快乐。

你将学到最基本的CSS、HTML、JavaScript，以及更进阶的VUE JQ react 。或者你也可以学习如何构建微信小程序。来吧，投入前端的怀抱😘😘。

后台

与前端不同，后台组的我们并不负责表面工作，我们只会在应用的背后默默支撑它的运行，作为用户交互和底层数据之间的桥梁。

对于一般用户而言，他们只能看到网站的外在是否好看，而在实际应用中，我们更为注重网站的内涵。如果一个网站在用户量稍多的情况下就卡死崩溃，外表再好看也无大用。（此处实名diss某校教务系统）

如果你对表面工作不感兴趣，想来点深♂入的，那么后台组欢迎你。

安卓

MAT实验室安卓部门以Java为基础语言，主要学习但不仅仅限于安卓手机APP的开发。

  • 学习后能干什么？学了就可以开发淘宝APP了(手动滑稽
  • 学习要求是什么？舍得学，别喊你学习都喊不动
  • 学习计划是什么？第一学期刷OJ打基础，第二学期带你入门Helloworld的App，从此走上人生巅峰。
  • 学习优势在哪里？安卓的学长最帅，学姐最漂亮。

欢迎各位对APP开发这方面有兴趣和没兴趣的帅哥美女加入我们。

硬件

MAT实验室硬件部主要通过对stm32的使用，stm32粗略的理解就是一块电路板，你可以自己在上面开发东西，实现各种特定的功能，以及硬件对软件的调控。

硬件设施的使用涉及到动手能力，所以有想自己动手做出个什么小设备的同学可以报名硬件部，硬件部充满了乐趣，但硬件部相对于别的部管也会比较严，希望对硬件感兴趣的小可爱可以报名硬件部。

大数据

当今社会各行各业每天都在产生大量数据，为了从海量数据中归纳整合出有价值的信息，大数据技术应运而生。

在这里，你能接触到网络数据采集，数据分析，数据图表可视化，把看似杂乱无章的数据以更加直观的形式呈现出来，从而得出结论，利用这些数据的价值。

在这里，你也能跟我们一起摸鱼，恶搞OJ，远程晚寝签到，1分钟刷完视听说，更过分的是利用脚本抢课（

加入大数据，一起学习，一起进步吧！

关于报名

看了以上的介绍，相信你也对各个方向有了基本的认识。当然，如果你突然有一天对选择的方向不满意，在进入实验室之后，你也可以更换方向。

请在这个页面填写你的学号并选择你最想去的方向，点击确定后页面将跳转到报名成功的列表。
